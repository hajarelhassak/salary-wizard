Salary Wizard - Prédiction de Salaire avec la Régression Linéaire

🔬 Description

Salary Wizard est un projet de Machine Learning qui prédit le salaire d'un employé en fonction de ses années d'expérience. Il utilise la régression linéaire via Scikit-Learn pour modéliser la relation entre l'expérience et le salaire.

📝 Données

Le dataset utilisé (salary_Data.csv) contient :

YearsExperience : Nombre d'années d'expérience

Salary : Salaire correspondant

🛠 Installation

Assurez-vous d'avoir Python 3.x installé sur votre machine.

Clonez le projet :

git clone https://github.com/hajarelhassak/salary-wizard.git
cd salary-wizard

Installez les dépendances :

pip install -r requirements.txt

🌟 Utilisation

Exécutez le script Python pour entraîner et visualiser le modèle :

python salary_prediction.py

Le programme :

Charge les données

Effectue une régression linéaire

Affiche le graphe avec la droite de régression

📊 Visualisation

Le script génère un graphique affichant la relation entre l'expérience et le salaire, avec la droite de régression superposée.
